0. System Requirements
Requisiti di Sistema
English

Before starting, verify your server has:

  • PHP 8.0+
  • MySQL 8.0+
  • Web Server: Apache or Nginx with mod_rewrite
  • Disk Space: 50 MB minimum
  • PHP Memory: 128 MB minimum
Required PHP Extensions:
mysqli (required), mbstring, json, fileinfo, gd (recommended)
Italiano

Prima di iniziare, verifica che il tuo server abbia:

  • PHP 8.0+
  • MySQL 8.0+
  • Web Server: Apache o Nginx con mod_rewrite
  • Spazio disco: 50 MB minimo
  • Memoria PHP: 128 MB minimo
Estensioni PHP Richieste:
mysqli (obbligatoria), mbstring, json, fileinfo, gd (consigliata)

2Create Database
Crea il Database
English
IMPORTANT: Database must be created before running the wizard.
Localhost (MAMP/XAMPP)
  1. Open phpMyAdmin (http://localhost/phpmyadmin)
  2. Click "New" in sidebar
  3. Database name: e.g. star51_db
  4. Collation: utf8mb4_unicode_ci
  5. Click "Create"
Shared Hosting (cPanel)
  1. Login to cPanel
  2. Go to "MySQL Databases" → "Create Database"
  3. Name: will be prefixed (e.g. user_star51)
  4. Write down full name, username and password
  5. Assign all privileges to user
Italiano
IMPORTANTE: Il database deve essere creato prima di eseguire il wizard.
Localhost (MAMP/XAMPP)
  1. Apri phpMyAdmin (http://localhost/phpmyadmin)
  2. Clicca su "Nuovo" nella sidebar
  3. Nome database: es. star51_db
  4. Collation: utf8mb4_unicode_ci
  5. Clicca "Crea"
Shared Hosting (cPanel)
  1. Login a cPanel
  2. Vai a "Database MySQL" → "Crea Database"
  3. Nome: verrà prefissato (es. user_star51)
  4. Annota nome completo, username e password
  5. Assegna tutti i privilegi all'utente
Fields Reference / Riferimento Campi:
Field / Campo Localhost Shared Hosting
Database Host localhost mysql.hosting.com
Database Name star51_db user_star51
Username root user_dbuser
Password root or empty From cPanel
3Run Setup Wizard
Esegui il Setup Wizard
English
  1. Open your browser
  2. Navigate to: http://your-domain.com/install/
  3. Fill in the installation form
Database Section

Enter the database credentials from Step 2. Use "Test Connection" button to verify.

Super Admin Section
  • First/Last Name: Your details
  • Email: Used for password reset
  • Username: For login
  • Password: Minimum 8 characters
Gmail SMTP (Optional)

With SMTP: Password reset via email enabled

Without SMTP: Temporary password generated, change after first login

Italiano
  1. Apri il browser
  2. Vai a: http://tuo-dominio.com/install/
  3. Compila il form di installazione
Sezione Database

Inserisci i dati del database creato nello Step 2. Usa il pulsante "Testa Connessione" per verificare.

Sezione Super Admin
  • Nome/Cognome: I tuoi dati
  • Email: Per il password reset
  • Username: Per il login
  • Password: Minimo 8 caratteri
Gmail SMTP (Facoltativo)

Con SMTP: Password reset via email attivo

Senza SMTP: Password temporanea generata, da cambiare al primo login

Gmail App Password Setup:
English
  1. Go to Google App Passwords
  2. Create a new App Password for "Mail"
  3. Copy the 16-character code
  4. Enter it in the "Gmail App Password" field
DO NOT use your regular Gmail password!
Italiano
  1. Vai su Google App Passwords
  2. Crea una nuova App Password per "Mail"
  3. Copia il codice a 16 caratteri
  4. Inseriscilo nel campo "Gmail App Password"
NON usare la tua password Gmail normale!
4Installation Complete
Installazione Completata
English
Success! If everything goes well, you'll see the success screen with login credentials.
Generated Files:
  • /nova/conf/nova_config_values.php
  • Database with 6 tables (ns_*)
  • Administrator account (level 0)
  • /.installed - Lock file
IMPORTANT - Security:
  1. Write down your credentials
  2. Delete /install/ folder for security:
    • Via Nova Admin → Settings → Box 7 (Installation Cleanup)
    • Or via FTP or terminal: rm -rf install/
Italiano
Successo! Se tutto va bene, vedrai la schermata di successo con le credenziali di login.
File Generati:
  • /nova/conf/nova_config_values.php
  • Database con 6 tabelle (ns_*)
  • Account amministratore (level 0)
  • /.installed - File lock
IMPORTANTE - Sicurezza:
  1. Annota le credenziali
  2. Elimina la cartella /install/ per sicurezza:
    • Via Nova Admin → Settings → Box 7 (Pulizia Installazione)
    • Oppure via FTP o terminale: rm -rf install/
5Login to Nova Admin
Login a Nova Admin
English

Login URL: http://your-domain.com/nova/
